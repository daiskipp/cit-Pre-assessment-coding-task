## Starting ##

```bash
# Clone this project
$ git clone https://github.com/daiskipp/cit-Pre-assessment-coding-task

# Access
$ cd cit_Pre-assessment-coding-task/variant3

# Run the project
$ node server.js

# The server will initialize in the <http://localhost:8000>
```
